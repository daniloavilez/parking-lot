# park-api [![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url] [![Dependency Status][daviddm-image]][daviddm-url]
> Parking lot API

## Installation

```sh
$ npm install --save park-api
```

## Usage

```js
const parkApi = require('park-api');

parkApi('Rainbow');
```
